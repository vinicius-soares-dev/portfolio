Descrição do projeto

Atualmente estou desenvolvendo esse projeto, que se trata de uma agenda pessoal onde você pode armazenar dados de seus contatos (nome, email, número de telefone, endereço). Nesse projeto estou utilizando as seguintes tecnologias: NodeJs (express), MongoDb (mongoose), EJS, CSS e JavaScript.
